# Data Dictionary

This document provides a data dictionary for the synthetic sales dataset used in this project. It describes each variable and its meaning.

| Column | Description |
| --- | --- |
| order_id | Unique identifier for each order |
| order_date | Date of the order |
| customer_age | Age of the customer at the time of purchase |
| gender | Gender of the customer (Male or Female) |
| country | Country where the order was placed |
| product_category | Category of the product purchased |
| quantity | Number of units purchased |
| unit_price | Price per single unit (USD) |
| total_price | Total price of the order (quantity × unit_price) |
| payment_method | Method of payment (e.g., Credit Card, PayPal, Cash) |
| customer_satisfaction | Customer satisfaction rating (1–5) |
| returned | Whether the order was returned (Yes or No) |
